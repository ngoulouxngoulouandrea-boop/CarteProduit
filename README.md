# CarteProduit

## Description

Page statique de catalogue produit pour une boutique écologique. La page affiche des cartes produits avec des badges de promotion et des boutons d’action.

## Fichiers

- `index.html` : page principale du catalogue.
- `styles.css` : styles CSS pour les cartes et la mise en page.
- Images : photos des produits référencées dans le HTML.

## Fonctionnalités

- Carte produit avec image, titre, description, évaluation et prix.
- Badges de stock/promotion.
- Boutons d’ajout au panier et de favoris.
- Utilisation de Google Fonts et Remix Icon.

## Utilisation

1. Ouvrir `index.html` dans un navigateur.
2. Vérifier que les images référencées sont présentes dans le dossier du projet.

## Notes

Cette page est un prototype statique ; aucun backend ou panier réel n’est implémenté.
